# System Patterns

- **UI/UX**: Notion-Style Sidepanels (Framer Motion) für Modale, Slide-in Animationen, fixierter Footer, Scrollbar-Breite Handling.
- **Data Access**: `useAdmin.ts` Hook oder direkte Supabase Client Calls im Service (`adminService.ts`).
- **Styles**: Tailwind CSS mit Shadcn UI Ansätzen. Toast-Notifications für Success/Error Feedbacks.
- **Routing**: `react-router-dom` in `App.tsx` und Protected Routes integriert in `AdminLayout.tsx`.
