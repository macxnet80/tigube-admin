# Project Brief

Das Tigube Admin Dashboard ist die zentrale Steuerungs- und Verwaltungsoberfläche für die Tigube Haustierbetreuungsplattform.
Das Ziel ist es, Administratoren alle notwendigen Tools zur Verfügung zu stellen, um Plattform-Inhalte (wie Blog, News, Werbeanzeigen, Hilfe-Center) sowie Benutzer und deren Abonnements zu verwalten.

## Kernziele
- Verwaltung aller Plattformdaten (CRUD-Operationen) für Content wie Hilfe-Center Ressourcen.
- Intuitive, moderne UI (React, Tailwind, Framer Motion für Sidepanels).
- Sichere Datenhaltung mit Supabase (PostgreSQL, RLS).
