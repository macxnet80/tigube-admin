# Project Context

Das Admin-Dashboard richtet sich an berechtigte Nutzer (Admins) von Tigube.
Es bietet eine Übersicht über Plattformstatistiken und Werkzeuge zur Verwaltung von Inhalten.

Neu hinzukommendes Feature: Hilfe-Center Verwaltung.
- Das Hilfe-Center enthält Tutorials, PDFs und Videos.
- Alle Ressourcen sollen öffentlich ("für alle User zugänglich") sein.
- Die Administrationsoberfläche benötigt eine CRUD-Schnittstelle.
